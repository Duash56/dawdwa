# Dummy Website

This repository contains a small **dummy static website** scaffolded for demos or testing.

## Files
- `index.html` — home page
- `about.html` — about page
- `contact.html` — contact form (dummy)
- `assets/css/styles.css` — styles
- `assets/js/main.js` — small interaction script
- `assets/images/placeholder.svg` — placeholder image

## Preview locally
- Open `index.html` in your browser, or run a simple local server:

  ```bash
  # from project root
  python -m http.server 8000
  # then open http://localhost:8000
  ```

- Or use VS Code Live Server extension.

## Deploy
- Push to GitHub and enable **GitHub Pages** from the repository settings (use `main` branch, root). Alternatively, use Netlify or Vercel for simple static hosting.
